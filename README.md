## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data structure and data types.
* Performed initial inspection of records and columns.

### 2. Exploratory Data Analysis (EDA)

* Analyzed dataset dimensions and summary statistics.
* Identified trends, distributions, and relationships.
* Visualized important metrics using charts and graphs.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Treated outliers where necessary.

### 4. SQL Analysis

* Loaded cleaned data into relational databases.

* Wrote SQL queries to:

  * Filter and aggregate data
  * Perform joins
  * Calculate KPIs
  * Generate business insights

* Queries were tested across:

  * PostgreSQL
  * MySQL
  * SQL Server

### 5. Dashboard Development

* Connected Power BI to the processed dataset.
* Built interactive dashboards and visual reports.
* Added filters, slicers, and KPI cards for user-friendly analysis.

### 6. Reporting & Presentation

* Created a business-focused analytical report.
* Summarized key findings and recommendations.
* Designed a presentation using Gamma AI for stakeholder communication.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL / MySQL / SQL Server
* Power BI Desktop

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/project-name.git
```

2. Navigate to the project directory:

```bash
cd project-name
```

3. Install required Python packages:

```bash
pip install pandas numpy matplotlib seaborn
```

4. Run the analysis notebook:

```bash
jupyter notebook
```

5. Execute SQL scripts in your preferred database environment.

6. Open the Power BI dashboard file (`.pbix`) to explore visualizations.
